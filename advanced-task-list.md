# 고급업무조회

`.claude/tasks/` 폴더에 저장된 고급업무 자료들을 조회하는 스킬입니다.

## 사용법

`/advanced-task-list [키워드]`

- 인자가 없으면: 모든 업무를 목록으로 표시
- 키워드 지정: 해당 키워드와 관련된 업무만 필터링하여 표시

## 수행 절차

### Phase 1: 자료 탐색

1. `.claude/tasks/` 폴더에서 업무계획서(`task_*.md`) 파일들을 탐색합니다.
2. 각 업무계획서를 읽어 다음 정보를 수집합니다:
   - 업무 제목
   - 상태 (진행중/완료)
   - 작성일
3. 각 업무의 관련 파일도 확인합니다:
   - TodoList (`todo_*.md`) 존재 여부 및 진행률 (완료/전체 항목 수)
   - 스크린샷 폴더 (`.claude/tasks/screenshots/{task_id}/`) 존재 여부 및 파일 수

### Phase 2: 결과 표시

다음 형식으로 사용자에게 보여줍니다:

```
## 고급업무 목록

| # | 업무 제목 | 상태 | 작성일 | Todo 진행률 | 스크린샷 |
|---|----------|------|--------|------------|---------|
| 1 | {제목}    | 진행중 | 2026-02-27 | 3/5 | 2개 |
| 2 | {제목}    | 완료   | 2026-02-25 | 4/4 | 없음 |

### 관련 파일
- 업무 1:
  - 계획서: .claude/tasks/task_20260227_xxx.md
  - TodoList: .claude/tasks/todo_20260227_xxx.md
  - 스크린샷: .claude/tasks/screenshots/20260227_xxx/ (2개 파일)
```

키워드가 지정된 경우, 업무 제목이나 내용에 키워드가 포함된 업무만 표시합니다.

## 중요 규칙

- 조회만 수행하며, 파일을 수정하거나 삭제하지 않습니다.
- 업무가 없으면 "등록된 고급업무가 없습니다."로 안내합니다.
- "진행중" 상태의 업무를 목록 상단에 배치합니다.
