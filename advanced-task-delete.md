# 고급업무삭제

완료된 업무의 문서와 관련 자료를 정리·삭제하는 스킬입니다.

## 사용법

`/advanced-task-delete [파일명 또는 키워드]`

- 인자가 없으면: `.claude/tasks/` 폴더의 모든 업무를 목록으로 표시
- 파일명/키워드 지정: 해당 업무를 직접 찾아 삭제 진행

## 수행 절차

### Phase 1: 업무 탐색 및 표시

1. `.claude/tasks/` 폴더에서 업무계획서(`task_*.md`) 파일들을 탐색합니다.
2. 각 업무의 제목, 상태(진행중/완료), 작성일을 목록으로 표시합니다.
3. 인자가 지정된 경우 해당 업무만 표시합니다.
4. 각 업무에 대해 관련 파일을 함께 표시합니다:
   - 업무계획서 (`task_*.md`)
   - TodoList (`todo_*.md`)
   - 스크린샷 폴더 (`.claude/tasks/screenshots/{task_id}/`) - 존재하는 경우

### Phase 2: 사용자 확인

1. 삭제 대상 업무를 사용자에게 보여줍니다.
2. 삭제 범위를 확인합니다:
   - 업무계획서 (`task_*.md`)
   - TodoList (`todo_*.md`)
   - 스크린샷 폴더 (`.claude/tasks/screenshots/{task_id}/`)
3. **반드시 사용자 승인을 받은 후** 삭제를 진행합니다.

### Phase 3: 삭제 수행

1. 승인받은 파일/폴더를 삭제합니다.
   - `rm` 명령으로 문서 파일 삭제
   - `rm -rf` 명령으로 스크린샷 폴더 삭제
2. 삭제 결과를 사용자에게 보고합니다.

## 중요 규칙

- **"진행중" 상태의 업무는 삭제 전 경고**: 미완료 작업이 있음을 알리고 재확인합니다.
- 삭제는 **되돌릴 수 없으므로** 반드시 사용자 확인 후 수행합니다.
- 여러 업무를 한번에 삭제할 수 있되, 각각에 대해 확인합니다.
- `screenshots/` 폴더가 비어있으면 폴더 자체도 삭제합니다.
